HTML 文本格式化

注意原则

        很多标签都可以用来改变文本的外观，并为文本关联其隐藏的含义。
        总地来说，这些标签可以分成两类：基于内容的样式（content-based style）和物理样式（physical style）。

        请注意，基于内容的样式会赋予页面更强的语义性，使浏览器可以更好地理解和处理页面，开发者应该多使用这些样式，它们包括
        <abbr>
        <acronym>(HTML5不支持)
        <cite>
        <code>
        <dfn>
        <em>
        <kbd>
        <samp>(很少用)
        <strong>
        <var>

        而物理标签大多只是控制文本的外观，这可以用样式表进行更好的处理，所以应该避免使用，它们包括 
        <b>
        <big>
        <i>
        <s>
        <small>
        <strike>
        <sub>
        <sup>
        <tt>

文本格式化标签

<b> 定义粗体文本 测试文字
<big> 定义大号字 测试文字
<em> 定义着重文字 测试文字
<i> 定义斜体字 测试文字
<small> 定义小号字 测试文字
<strong> 定义加重语气 测试文字
<sub> 定义下标字 测试文字
<sup> 定义上标字 测试文字
<ins> 定义插入字 测试文字
<del> 定义删除字 测试文字

“计算机输出标签”

<code> 定义计算机代码 printf('hello world!')
<kbd> 定义键盘码 Shift
<samp> 定义计算机代码样本 sample
<tt> 定义打字机代码 hello world
<var> 定义变量 hello world
<pre> 定义预格式文本（保留空格和缩进） 
                    hello world!
                      can I help you?

引用、引用和术语定义

<abbr> 定义缩写 Dota is a moba game
<address> 定义地址
#20A, 2345 Belmont Avenue, Durham, NC， 27700
<bdo> 定义文字方向(ltr/rtl) Here is some text
<blockquote> 标记长的引用
请注意，浏览器在 blockquote 元素前后添加了换行，并增加了外边距。
<q> 定义短的引用语 <q> 标签用于简短的行内引用。
<cite> 定义引用、引证 <cite>标签通常表示它所包含的文本对某个参考文献的引用，比如书籍或者杂志的标题。如：Playboy
<dfn> 定义一个定义项目 <dfn> 标签可标记那些对特殊术语或短语的定义。如：fast Fourier transform
